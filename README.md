# websocket demo

### 效果图
![image](https://github.com/feiaaa/demos/blob/master/websocketDemo.gif) 

### 安装方法
请先确保装了node 和npm
```
npm install

```
### 运行方法
用cmd 打开，找到文件所在路径
```
node server.js
```
然后打开client.html


### 说明
- 本项目只是websocket下面的一个例子，更多请参考[这里](https://github.com/websockets/ws)

- 参考文章[用nodejs搭建websocket服务器](https://www.jb51.net/article/104100.htm)
